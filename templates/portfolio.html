Projects

Here are some of the projects I’ve worked on using technologies like Flutter, C#, Firebase, and Python. My focus is on mobile and web development, data-driven applications, and user-friendly interfaces.

Pong Game

A simple yet fun Pong game built in Python using the Pygame library. Players control paddles to bounce the ball and try to score points. The game includes a scoring system and adjustable game speed, providing an interactive and nostalgic gaming experience.

eCommerce Website for iPhones & Sneakers

Developed a responsive eCommerce website using Flutter Web to showcase iPhones and sneakers. The platform allows users to browse products, filter by category, and contact sellers directly without a checkout process.

Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using Python. The game allows two players to play against each other on the same device, featuring a command-line interface for easy play.

Exam Preparation App

A Flutter-based mobile app designed to help high school learners prepare for exams. It features categorized past questions (MCQs, Terminology, Matching), difficulty levels, exam tips, and subject-focused insights.

Beauty eCommerce Site

A Flutter web catalog for premium weaves, braids, and makeup bookings. Supports filtering, WhatsApp contact, and multi-category services.

Snake Game

A classic Snake game built using Python. The player controls the snake to eat food, grow longer, and avoid collisions with the walls or itself. Includes simple controls and a scoring system.
